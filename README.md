## Project setup

```
composer install
```

### Install jetstream

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Migrate database

```
php artisan migrate
```

### Start server

```
php artisan serve
```
