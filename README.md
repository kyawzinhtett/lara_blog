## Project setup
### Install dependencies

```
composer install
```

### Install jetstream

```
npm install
```

```
npm run build
```

### Generate key

```
php artisan key:generate
```

### Migrate database

```
php artisan migrate
```

### Start server

```
php artisan serve
```

## Features

### Profile Panel

-   Posts Management (CRUD)

### Posts Panel

-   View Public Posts
-   Comments
